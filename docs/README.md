### 📊 RCR_Metric_Design.md
Documentación técnica de la **Métrica RCR (Reflex Coherence Ratio)** - la cuantificación de coherencia ontológica.

**Fórmula:**
```
RCR = (PCO - PE) / (PCO + PE) × 100
```

Donde:
- **PCO:** Puntuación de Coherencia Ontológica
- **PE:** Puntuación de Entropía
